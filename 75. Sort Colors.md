# [75. Sort Colors](https://leetcode.com/problems/sort-colors/)
## python
### 解
* 解法：
    ```python
    def merge(self, intervals):
        intervals.sort(key=lambda x: x[0])
        res = [intervals[0]]
        for interval in intervals[1:]:
            if res[-1][1] >= interval[0]:
                res[-1][1] = max(interval[1], res[-1][1])
            else:
                res.append(interval)
        return res
    ```
